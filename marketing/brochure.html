Quantum-Safe Security Architecture

Every layer of QS-VC's media pipeline is protected with post-quantum cryptography — from signaling to media frames to recordings at rest.

1

Signaling Security (Quantum-TLS)

Hybrid Kyber-1024 + X25519 key exchange, Dilithium-5 + Ed25519 signatures, TLS 1.3 with PQC extensions

2

Media End-to-End Encryption (E2EE)

SFrame via Insertable Streams, per-participant Kyber keypairs, frame-level AES-256-GCM, zero-knowledge SFU

3

Key Management

HSM-backed (Shamir 5-of-8), 60-second media key rotation, hourly TURN credential rotation

4

Recording Encryption

Kyber-wrapped Data Encryption Keys, encrypted at-rest storage (MinIO/S3), tamper-proof metadata

5

Audit Trail

Merkle-tree verified hash chains, Dilithium-signed audit entries, configurable retention
